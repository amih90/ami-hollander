## Why a Personal Site?

After years of building tools, contributing to open-source projects, and helping developers ship their best work, I realized something: I had no *home base* on the internet. Sure, there's GitHub, LinkedIn, and the occasional conference talk — but nothing that felt like *mine*.

So I built this. A place to share what I'm working on, what I've learned, and what I'm thinking about.

## What to Expect

Here's the changelog for this blog:

- **Cloud & Infrastructure** — Deep dives into Azure, containers, serverless, and the tools that power modern development.
- **AI & Developer Tools** — From MCP servers to AI-assisted coding, I'll share what's working and what's hype.
- **Open Source** — Contributions, lessons learned, and why giving back to the community matters.
- **Career & Leadership** — Thoughts on growing as an engineer, mentoring, and building great teams.

## The Stack

This site is built with:

- **Next.js** + **TypeScript** — React with server-side rendering
- **Tailwind CSS v4** — Utility-first styling with custom pixel/terminal theme
- **Azure Static Web Apps** — Serverless hosting with global CDN
- **MDX** — Markdown with component superpowers for blog posts

The source code is open on [GitHub](https://github.com/amih90/ami-hollander). Feel free to fork it or file issues.
